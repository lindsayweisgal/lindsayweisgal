Commonly Used HTML Tags

html
Outline Tags
Tag Description
!DOCTYPE html Defines document type
Defines the root of an HTML document
head Contains metadata for the document
title Document title, shows up in tab
body Document body/content
footer Footer
!-- -- Comment
General Tags
Tag Description
div Defines a section
h1 to h6 Headings
p Paragraph
a Hyperlink
img Embeds image
audio Embeds audio
video Embeds video
Figure Specifies self-contained content
link Links a document and an external resource (e.g. style sheets)
figcaption Caption for a figure element
button Clickable button
aside Content aside from page content
br Line break
span Defines a section in a document
style Defines style information for a document
ol Numbered list
ul Un-ordered list
li List item
nav Navigation link
Table Tags
Tag Description
table The wrapper element for all HTML tables.
thead Groups the header content in a table
tbody The set of rows containing actual table data.
th Defines a header cell in a table
tr The table row container.
td Defines a cell in a table
tfoot The set of rows defining the footer in a table.
Formatting Tags
Tag Description
em Emphasized (italic) text
b Bold text
strong Bold text, also indicating high importance
small Reduce text size
mark Highlighted text
sub Subscripted text
sup Superscripted text
